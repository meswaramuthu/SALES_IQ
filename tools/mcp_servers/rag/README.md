# stratova-rag-mcp
RAG cache check/write + semantic search over shared Vertex AI RAG corpus

## Environment Variables
- `KNOWLEDGE_IQ_RAG_CORPUS`
- `GOOGLE_CLOUD_PROJECT`
- `GOOGLE_CLOUD_LOCATION`

## Deploy
```bash
bash deploy.sh
```

## Local development
```bash
pip install -r requirements.txt
python server.py
```
